В серсивы мы выносим логику оброботки данных

В mail-service мы используем либу nodemailer
Nodemailer — это модуль для приложений Node.js, позволяющий легко отправлять электронные письма.
В createTransport - нам нужно иницылизировать данные для отправки

И отправляем письмо через метод sendMail



В token-service мы используем либу jsonwebtoken 
Jsonwebtoken - Реализация веб-токенов JSON
sign - принимает данные и secret key, по которому шифрует информацию
expiresIn - время жизни токена


В user-service мы импользуем либы bcrypt, uuid
bcrypt - чтобы захешировать пароль ( чтобы в дб пароль хранился  виде хеша )
uuid - создает id для активации почты

Мы получаем email, password проверяем если ли такой пользователь в бд
и создаем нового
