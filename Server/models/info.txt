Описание схем для mongodb
Через которые мы обращаемся к нашей бд и обрабатываем, сохраняем, удалеем и тд.

Нам нужно описать типы полей нашей сущности ( одного объекта грубо говоря )

unique: true - не повторяющейся
required: true - обязательное поле
default - значения по умолчанию
Schema.Types.ObjectId, ref: "User" - тип поля будет наш другой объект на который ссылается ref
По умолчанию Mongoose добавляет _id свойство в ваши схемы.
